AI Text & Image Generator Web App
This project is a simple web-based tool that combines text enhancement and image generation
using AI APIs.
Features:
1. Text to Image Workflow:
- User enters a prompt
- The system enhances the prompt
- User reviews and generates an image
2. Image Upload & Analysis:
- Upload image
- Analyze content
- Suggest variations
Tech Stack:
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
APIs: OpenAI API
Setup:
1. Clone repo
2. Install backend dependencies
3. Add API key in .env
4. Run backend
5. Open frontend
Notes:
- Ensure API key is valid
- Backend must be running
Conclusion:
This project demonstrates API integration and AI workflows in a simple web app.
